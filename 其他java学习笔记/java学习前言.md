# 第二个：快捷键

- win+e：打开我的电脑
- ctrl+shift+esc：打开资源管理器
- alt+F4：关闭当前窗口
- 资源管理器是桌面，关闭后在启动就是explorer
- 在文件夹下按住shift+右键点击，打开控制台

# 第三个：DOS命令

- E:——切换到e盘
- calc——打开计算器
- mspaint——打开画图工具
- notepad——打开记事本
